00141    * (a) Pre-processing involves multiplying input with cos factor,

00148    *                  Step2: Calculating FFT of the re-ordered input.   
00149    *                  Step3: Taking the real part of the product of FFT output and weights.   
00150    * (c) Post-processing - DCT4 can be obtained from DCT2 output using the following equation:   
00151    *                   Y4(k) = Y2(k) - Y4(k-1) and Y4(-1) = Y4(0)   
00152    *                        where,   
00153    *                           Y4 -- DCT4 output,   Y2 -- DCT2 output   
00154    * (d) Multiplying the output with the normalizing factor sqrt(2/N).   
00155    */
