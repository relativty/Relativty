00097 * Radix-4 FFT algorithm used is :   
00098 *   
00099 * Input real and imaginary data:   
00100 * x(n) = xa + j * ya   
00101 * x(n+N/4 ) = xb + j * yb   
00102 * x(n+N/2 ) = xc + j * yc   
00103 * x(n+3N 4) = xd + j * yd   
00104 *   
00105 *   
00106 * Output real and imaginary data:   
00107 * x(4r) = xa'+ j * ya'   
00108 * x(4r+1) = xb'+ j * yb'   
00109 * x(4r+2) = xc'+ j * yc'   
00110 * x(4r+3) = xd'+ j * yd'   
00111 *   
00112 *   
00113 * Twiddle factors for radix-4 FFT:   
00114 * Wn = co1 + j * (- si1)   
00115 * W2n = co2 + j * (- si2)   
00116 * W3n = co3 + j * (- si3)   
00117 *   
00118 *  Butterfly implementation:   
00119 * xa' = xa + xb + xc + xd   
00120 * ya' = ya + yb + yc + yd   
00121 * xb' = (xa+yb-xc-yd)* co1 + (ya-xb-yc+xd)* (si1)   
00122 * yb' = (ya-xb-yc+xd)* co1 - (xa+yb-xc-yd)* (si1)   
00123 * xc' = (xa-xb+xc-xd)* co2 + (ya-yb+yc-yd)* (si2)   
00124 * yc' = (ya-yb+yc-yd)* co2 - (xa-xb+xc-xd)* (si2)   
00125 * xd' = (xa-yb-xc+yd)* co3 + (ya+xb-yc-xd)* (si3)   
00126 * yd' = (ya+xb-yc-xd)* co3 - (xa-yb-xc+yd)* (si3)   
00127 *   
00128 */
